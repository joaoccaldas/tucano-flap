## What Got Built (v0.1)

### Core Game
- **Tucano** (orange toucan) with physics: gravity, flap impulse, rotation
- **Pipes** spawn every 2 seconds with randomized gaps
- **Collision detection** — hit pipes, ground, or ceiling = game over
- **Score tracking** with high score persistence in memory
- **Particle effects** — white puffs on flap, gold sparkles on score
- **Game states**: Menu → Playing → Paused → Game Over → Restart

### Controls (Samsung TV)
- **Enter/OK**: Flap / Start game / Restart
- **Escape/Back**: Pause / Resume
- Also works with mouse/touch for testing

### Tech
- TypeScript + HTML5 Canvas
- Vite build system
- 1920x1080 TV-optimized resolution
- 60fps game loop with delta time
